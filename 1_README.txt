This is the instruction about how to run automated tests.

Note: Be sure to turn configuration of users into original state after each manual test case, because changed configuration may affect test results of automated tests.

If you are a Windows user:
1. Open mvnJasperReports folder
2. Run win_1_clean.bat
3. Run win_2_compile.bat
4. Run win_3_test-compile.bat
5. Run win_4_test.bat

If you are an OS X user:
1. Open mvnJasperReports folder
2. Run OSx.sh
